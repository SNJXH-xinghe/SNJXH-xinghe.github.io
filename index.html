平台简介

本平台结合华为 Health Kit 数据与 AI 驱动的 DeepSeek 分析，
支持用户上传或 API 拉取健康数据（JSON/XML/CSV），即时生成可下载的 CSV 预览，
并基于个人基础疾病标签和饮食偏好，自动推荐营养均衡的食谱方案。
同时，配合健康顾问模块，提供运动与生活方式建议，帮助您实现全面的健康管理。

核心功能

多格式数据接入

支持 JSON / XML / CSV 文件上传，或 OAuth2 API 一键拉取

即时数据预览

上传后实时渲染 CSV 表格，并提供下载链接

健康风险评估

结合已选基础疾病标签，评估代谢、心血管等多系统风险

个性化食谱推荐

根据用户偏好与健康状况，生成多道营养食谱

运动 & 生活方式建议

基于健康数据提供运动处方与日常健康指导

互动循环查看

“下一道菜”按钮，可循环浏览推荐菜谱
